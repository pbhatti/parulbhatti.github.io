Image to 3D Web AR App

Vibecoded a mini web-based app that converts any 2D sketch into a 3D model that can be spawned in the AR environment, powered by the Meshy Image-to-3D API.


TECH STACK

• Frontend:  Vanilla JavaScript (Vite), Three.js for WebXR (Android), AR Quick Look (iOS)

• Backend:  Node.js + Express (Meshy API proxy)

• API:  Meshy Image-to-3D (GLB + USDZ outputs)


STATUS

Ongoing

CHROME ON IOS
SAFARI ON IOS
Features

• Launch & Permission: One-tap start with system camera access prompts.

• Capture Mode: Point and capture, send to Meshy for 3D generation, view results in AR.

• Gallery Mode: Browse and relaunch past generated 3D models.
